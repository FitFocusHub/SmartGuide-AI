# SmartGuide AI - Desktop Installer

## Installation

1. Double-click `SmartGuideAI_Setup.exe`
2. Click **Install**
3. Done! Find "SmartGuide AI" in Start Menu

## Uninstall

1. Go to Start Menu
2. Find "SmartGuide AI"
3. Click "Uninstall"

Or delete: `%APPDATA%\SmartGuideAI` folder

## Requirements

- Windows 10/11
- Python 3.8+ (for AI features)
- Internet connection

## What's Included

- Desktop Server (WebSocket)
- Browser Extension
- Mobile PWA App

## Quick Start

1. Install using this .exe
2. Start "SmartGuide AI" from Start Menu
3. Load browser extension in Chrome
4. Start asking questions!
