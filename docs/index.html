SmartGuide AI v1.0

Chrome Extension Python Server AI Powered

Installation

1 Download Installer

Click below to download the one-click installer:

Download SmartGuideAI-Setup.exe

2 Run Installer

Double-click the downloaded file and follow the wizard:

SmartGuideAI-Setup.exe
Note: Server will start automatically after installation!

3 Load Chrome Extension

Open Chrome and follow these steps:

  • Type in address bar:
chrome://extensions/
  • Enable Developer mode (top right)
  • Click Load unpacked
  • Select this folder:
C:\Program Files\SmartGuide AI\browser-extension

4 Get Free API Key

Go to Groq console and create a free API key:

Get API Key from console.groq.com

5 Add API Key

  • Click SmartGuide icon in Chrome toolbar (puzzle piece)
  • Paste your API key
  • Click Save

6 Use SmartGuide!

  • Go to any website (YouTube, ChatGPT, etc.)
  • Click the purple chat bubble
  • Ask your question!

Alternative Installation

Option 2: pip install

pip install smartguide-ai
smartguide-server

Option 3: Manual Setup

git clone https://github.com/FitFocusHub/SmartGuide-AI.git
cd SmartGuide-AI
pip install -r server/requirements.txt
python server/server.py

Troubleshooting

"Server Offline" in extension?

Run this command to start server:

cd "C:\Program Files\SmartGuide AI" && start_server.bat
"No API Key" error?

Get free key from:

https://console.groq.com
Requirements:
  • Python 3.8+
  • Chrome/Edge/Brave browser
  • Free Groq API key
