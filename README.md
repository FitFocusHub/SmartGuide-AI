# 🤖 SmartGuide AI

Real-time AI Guidance for Any Software.

## Features

- Chat with AI about any software
- Auto-highlight where to click
- Control desktop apps with voice
- YouTube, Excel, CapCut, WhatsApp support
- Hinglish responses

## Download & Install

### Windows (.exe)

**Coming Soon!**

Build your own:
```
1. Download this repo
2. Open installer folder
3. Run build_exe.bat
4. Get SmartGuideAI_Setup.exe
```

### Android (APK)

**Coming Soon!**

Build your own:
```
1. Download this repo
2. Open mobile-app folder
3. Run build_apk.bat
4. Get SmartGuideAI.apk
```

### Browser Extension

```
1. Download this repo
2. Open Chrome → chrome://extensions/
3. Enable Developer mode
4. Click Load unpacked → Select browser-extension folder
```

## Quick Start

```
1. pip install -r requirements.txt
2. python main.py (start server)
3. Load extension in Chrome
4. Start asking questions!
```

## Commands

- `"Notepad kholo"` - Opens Notepad
- `"Excel kholo"` - Opens Excel  
- `"Video chalao"` - Play/Pause YouTube
- `"10 min aage"` - Seek forward
